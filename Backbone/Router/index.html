Backbone - Router

The Backbone router is used to invoke functions when certain hash fragments are appended to the main browser URL. This is useful for providing navigation within a single-page application as well as for providing support for bookmarks.

This sample demonstrates a single page application with two views that can be bookmarked. Navigation is achieved both with on-page links as well as in response to button clicks. A browser history is built and maintained as the navigation occurs.
